# Radio

## Usage
<ExampleContainer>
    <FilterProvider filter={categoryFilter()}>
        <ul>
            <li><Radio name="All" /> <label>All</label></li>
            <li><Radio name="Blog" /> <label>Blog</label></li>
        </ul>
    </FilterProvider>
</ExampleContainer>

```javascript
import { Filter, FilterProvider, Radio } from "@sajari/sdk-react";

const filter = new Filter({
        "All": "",
        "Blog": "dir1='blog'",
    }, 
    ["All"]
)

<FilterProvider filter={filter}>
    <ul>
        <li><Radio name="All" /> <label>All</label></li>
        <li><Radio name="Blog" /> <label>Blog</label></li>
    </ul>
</FilterProvider>
```
