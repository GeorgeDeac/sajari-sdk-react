# Input

The `Input` component provides a text-box which performs searches as the user types.
It is customizable through the use of props, options include rendering autocomplete
suggestions directly in the input box, or alternatively in
a drop-down list underneath.

```javascript
// Import the Input component
import { Input } from "@sajari/sdk-react"
```

## Usage

### With default props
<Playground>
    <Input />
</Playground>


### With instant search and typeahead
<Playground>
    <Input mode="typeahead" instantSearch />
</Playground>


### With suggestions dropdown
<Playground>
    <Input mode="typeahead" dropdownMode="suggestions" />
</Playground>



### With custom styles
<Playground>
    <Input mode="typeahead" dropdownMode="suggestions" styles={{
        container: {
            width: "300px",
            marginBottom: 0
        },
        input: {
            container: {
                backgroundColor: "#999",
                color: "#fff",
                border: "1px solid #999",
                boxShadow: "none"
            },
            button: {
                color: "#eee"
            }
        }
    }} />
</Playground>
