# Sajari Search Overlay

## Set-up

```bash
git clone https://github.com/sajari/sajari-sdk-react/
cd sajari-sdk-react/examples/overlay
npm install
```

## Run

Opens the app in your browser with hot reloading enabled.

```bash
npm start
```

## Build

Outputs a built static site in `build/`.

```bash
npm run build
```
