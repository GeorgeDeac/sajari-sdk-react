Project: {{.project}} Collection: {{.collection}}

This example injects a search interface into a page.

It includes a minimal set of styles, code and config to get you up and running:

  • Use query param ?q=set+search+using+query+param to set the default search. Give it a try!
  • Search UI is injected into <div id="search-interface"></div>

Take a look at the page source to start your own integration.

Here are two example points of integration.

1. From a user filled text box

2. From a user focusing a text box or clicking a button
